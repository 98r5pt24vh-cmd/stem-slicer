# Stem Slicer 1.0

Windows build project for Stem Slicer 1.0.

The Windows executable is built by GitHub Actions. FFmpeg and FFprobe are bundled into the executable during the build, so users do not need to install them separately.

## Build

Push this repository to GitHub, then open the `Actions` tab and run the Windows build workflow.

The generated `.exe` is uploaded as a workflow artifact.
